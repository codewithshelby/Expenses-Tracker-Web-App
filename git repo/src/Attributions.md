 # ABOUT 
      SPENDLY IS A FULL-FEATURED PERSONAL FINANCE MANAGEMENT WEB APPLICATION BUILT BY OUR TEAM AS PART OF THE DIPLOMA IN CSE PROGRAMME AT GOVERNMENT POLYTECHNIC PANCHKULA. THE PROJECT WAS DEVELOPED USING REACT 18, TYPESCRIPT, TAILWIND CSS, RECHARTS, FRAMER MOTION, AND LUCIDE REACT. IT IS DESIGNED TO HELP STUDENTS AND YOUNG PROFESSIONALS TRACK DAILY EXPENSES, LOG INCOME, MANAGE MONTHLY BUDGETS, AND VIEW SPENDING PATTERNS THROUGH INTERACTIVE CHARTS.

# PURPOSE
     THE PROJECT WAS AIMED AT GAINING PRACTICAL EXPERIENCE IN MODERN WEB DEVELOPMENT THROUGH BUILDING A REAL, USABLE APPLICATION. WORKING ON ALL MODULES – AUTHENTICATION, DASHBOARD, EXPENSE TRACKING, INCOME LOGGING, BUDGET MANAGEMENT, AND REPORTS – HELPED US STRENGTHEN OUR UNDERSTANDING OF REACT COMPONENTS, TYPESCRIPT, CONTEXT API STATE MANAGEMENT, AND DATA VISUALISATION USING RECHARTS.

# KEY CODE FUNCTION.
  • APPCONTEXT.TSX – GLOBAL STATE PROVIDER WITH CRUD METHODS FOR EXPENSES, INCOME, AND BUDGETS, EXPOSED VIA THE USEAPP() CUSTOM HOOK.
  • DASHBOARD.TSX – COMPUTES 4 STAT CARDS AND RENDERS AREACHART AND PIECHART USING USEMEMO FOR DERIVED MONTHLY DATA.
  • EXPENSES.TSX – REAL-TIME SEARCH, CATEGORY FILTER, MONTH FILTER, SORT, INLINE ADD/EDIT FORM, AND DELETE CONFIRMATION.
  • BUDGET.TSX – COMPUTES SPENT, PERCENTAGE, AND STATUS (OK/WARNING/OVER) PER CATEGORY USING USEMEMO ON CURRENT MONTH EXPENSES.
  • REPORTS.TSX – USER-SELECTABLE CHART TYPE (BAR, AREA, PIE) AND DATE RANGE (3, 6, 12 MONTHS).
